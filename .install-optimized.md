# Optimized Installation Guide

## Quick Install (Recommended)

The dependencies have been optimized to reduce install time:

1. **Only essential wallet adapters** - Phantom and Solflare only (removed Torus and wallet-adapter-wallets package)
2. **Removed axios** - Using native fetch API instead
3. **Removed ESLint** - Can be added later if needed
4. **Minimal dev dependencies** - Only what's required

## Install Command

```bash
npm install --legacy-peer-deps
```

The `--legacy-peer-deps` flag helps avoid dependency conflicts with Solana packages.

## Alternative: Use npm ci (faster)

If you have a package-lock.json:
```bash
npm ci --legacy-peer-deps
```

## If Install Still Takes Too Long

### Option 1: Use Yarn (often faster)
```bash
yarn install
```

### Option 2: Use pnpm (fastest)
```bash
npm install -g pnpm
pnpm install
```

### Option 3: Install without optional dependencies
```bash
npm install --no-optional --legacy-peer-deps
```

## Expected Install Time

- **Before optimization**: ~5-10 minutes (many peer dependencies)
- **After optimization**: ~2-3 minutes (minimal dependencies)

## Troubleshooting

If you encounter issues:

1. **Clear npm cache**:
   ```bash
   npm cache clean --force
   ```

2. **Delete node_modules and reinstall**:
   ```bash
   rm -rf node_modules package-lock.json
   npm install --legacy-peer-deps
   ```

3. **Check Node.js version** (should be 18+):
   ```bash
   node --version
   ```

## Adding More Wallets Later

If you need more wallets, install them individually:
```bash
npm install @solana/wallet-adapter-sollet --legacy-peer-deps
```

Then add to `WalletContext.tsx`:
```typescript
import { SolletWalletAdapter } from '@solana/wallet-adapter-sollet'
// Add to wallets array
```

This approach is better than installing the entire `@solana/wallet-adapter-wallets` package.
